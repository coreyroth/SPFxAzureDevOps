## azure-dev-ops

This is a sample project used to demonstrate CI / CD with Azure DevOps. 

Read more in the blog post
https://coreyroth.com/2019/05/06/how-to-setup-ci-cd-in-azure-devops-with-the-help-of-spfx-generator/

### Building the code

```bash
git clone the repo
npm install
gulp serve
```
